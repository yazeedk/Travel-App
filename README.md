# Travel-App

### Navigate to the travelPlanner directory:
``` cd travelplanner ```

### Initialize the project:
``` npm init ```

### Install Webpack Dev Server:
``` npm install --save-dev webpack-dev-server ```

### Run the application:
``` npm start -- --port 3001 ```
OR
``` npm start ```
